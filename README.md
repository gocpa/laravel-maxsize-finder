Пакет с командой, которая показывает большие файлы в хранилище s3

## Установка

You can install the package via composer:

```bash
composer require gocpa/laravel-maxsize-finder
```

## Использование

```bash
php artisan gocpa:laravel-maxsize-finder
```

## Testing

```bash
composer test
```
